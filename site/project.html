Описание проекта

Проект представляет собой модернизацию и редизайн сервисов, связанных с проектной деятельностью в Московском Политехническом Университете.

Основные компоненты системы:

  • Витрина проектов - платформа, где студенты выбирают проект, на который они записываются
  • Личный кабинет студента - интерфейс для отслеживания своей работы по проектной деятельности
  • Личный кабинет куратора проекта - инструмент для управления проектом, выставления баллов и отслеживания посещаемости

Задачи проекта

Исследовательская часть

  • Анализ ключевых недостатков имеющийся цифровой инфраструктуры
  • Поиск и сравнение внешних сервисов для заимствования системы сортировки
  • Детальный анализ Procallab

Прототипирование

  • Создание прототипов Личного кабинета студента и преподавателя
  • Разработка макета витрины проектов в Figma
  • Тестирование UX/UI решений
